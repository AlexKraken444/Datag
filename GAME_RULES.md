# Правила игры

## Команды

Матч — **2 на 2**. Всего 4 игрока:

| Команда A | Команда B |
| --------- | --------- |
| Tager A   | Tager B   |
| Lighter A | Lighter B |

Победа — у команды, первой набравшей **12 очков**.

## Роли

### Tager
- Свободно перемещается по арене.
- Управление: `WASD` — движение, `Shift` — спринт (расходует стамину).
- Цель — наступить на **тень вражеского Tager-а**.

### Lighter
- Заперт в своей зоне (синие/красные полосы по краям арены).
- Управление: `A` / `D` — движение вдоль зоны, мышь — направление прожектора, **ЛКМ** — увеличить яркость, **ПКМ** — уменьшить.
- Цель — освещать союзного Tager-а так, чтобы вражеский Tager не мог наступить на нашу тень, и наоборот — стараться подсветить так, чтобы союзный Tager оказался в выгодной позиции для атаки на тень врага.

## Поле

- Квадратная арена 1000×1000 px (вид сверху).
- В двух противоположных углах — стартовые зоны Tager-ов (прямоугольные треугольники).
- Вдоль внешних сторон арены — зоны Lighter-ов (узкие прямоугольники).

## Начало раунда

1. Tager A / Tager B спавнятся в своих стартовых зонах.
2. Lighter A / Lighter B спавнятся в своих зонах.
3. Идёт отсчёт **3 → 2 → 1 → GO**.
4. Lighter могут целиться уже во время отсчёта; Tager-ы начинают двигаться только после `GO`.

## Тени — формула

На каждом тике (60 Гц) сервер пересчитывает теневой полигон:

```
direction = normalize(tagerPos - lightPos)
perp      = perpendicular(direction)
left      = tagerPos + perp * R
right     = tagerPos - perp * R
length    = clamp(BASE * brightness * (1 + dist/DIST_SCALE) * (0.4 + 0.6 * aimAlign),
                  MIN_LENGTH, MAX_LENGTH)
shadowQuad = [left, left + dir*length, right + dir*length, right]
```

Параметры (`lib/constants.ts`):

- `R = 18` — радиус Tager-а
- `BASE = 80`, `MIN = 24`, `MAX = 520`
- `DIST_SCALE = 600`
- `CONE_ANGLE = 45°` (для отрисовки конуса света)
- `brightness ∈ [0.4 .. 1.6]`, изменяется ±0.6/сек при удержании ЛКМ/ПКМ
- `aimAlign ∈ [0..1]` — насколько прожектор смотрит прямо на Tager-а

Тень и попадание считаются **алгоритмом point-in-polygon** на стороне сервера.

## Очки

| Событие                                                                    | Очки     |
| -------------------------------------------------------------------------- | -------- |
| Tager наступил на тень противника                                          | **+1**   |
| Tager наступил на тень противника, **пока противник в своей стартовой зоне** | **+2** (бонус) |
| Оба Tager-а одновременно наступили друг на друга (**≤ 100 мс**)            | ничья, никому |

После начисления очка раунд завершается. Через `POST_ROUND_DELAY_MS = 1500 мс` начинается новый раунд (3-2-1-GO).

## Победа / окончание матча

- Матч заканчивается, когда команда A или B первой достигает **12 очков**.
- Показывается экран победы с финальным счётом и индивидуальной статистикой попаданий каждого Tager-а.
- Можно сразу запустить реванш в той же комнате (счётчик готовности сбрасывается).

## Безопасность

Стартовая зона **не** является убежищем: тень там делает попадание **бонусным** для атакующего. Поэтому стоять в углу всю игру невыгодно.
